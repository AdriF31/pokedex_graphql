# Pokedex
Aplikasi untuk menampilkan daftar pokemon yang didapat dari[(https://beta.pokeapi.co/graphql/v1beta)](https://beta.pokeapi.co/graphql/v1beta)

## Feature
* GetX
* MVC
* Lottie Animation
* Infinite Scroll Pagination
* graphql
* hasura connect
